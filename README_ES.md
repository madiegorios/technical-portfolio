# Mario Adiego — Portfolio Técnico

**Ingeniero Informático | Ingeniería de Software | Automatización con IA | Gestión Técnica de Proyectos**

Este portfolio presenta áreas de trabajo y casos de estudio técnicos en un formato seguro y de alto nivel. Su objetivo es demostrar cómo analizo sistemas software, decisiones de producto, oportunidades de automatización y documentación técnica.

## Perfil

Combino experiencia en desarrollo de software con visión de producto y negocio. Mis áreas más fuertes son el desarrollo de software empresarial, bases de datos relacionales, aplicaciones Java, SQL, automatización de flujos de trabajo y desarrollo asistido por inteligencia artificial.

Me interesan especialmente roles como:

- Technical Project Manager
- Solutions Engineer
- Especialista en automatización con IA
- Consultor técnico
- Software Engineer con responsabilidades de producto

Este repositorio no incluye código privado, credenciales, esquemas de base de datos, arquitectura interna, prompts de producción ni detalles sensibles de implementación. Está enfocado en comunicar razonamiento técnico y madurez de proyecto sin exponer activos que deben mantenerse privados.
