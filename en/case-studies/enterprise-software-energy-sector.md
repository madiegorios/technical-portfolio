# Enterprise Software — Energy Sector Case Study

## Context

This case study summarizes high-level professional experience contributing to enterprise software systems in the energy sector. It is intentionally anonymized and does not include proprietary information, client data, internal architecture, credentials, or production details.

## Environment

The work involved corporate software development practices, structured task management, version control, deployment processes, and collaboration inside multidisciplinary teams.

## Technical areas involved

- Java-based enterprise application development
- SQL queries over relational databases
- Web technologies including HTML and JSP
- Code maintenance and feature implementation
- Version control and deployment support using Bitbucket
- Task tracking and project coordination using Jira
- Daily development using Eclipse IDE and Visual Studio Code

## Main responsibilities

- Contributing to the development and maintenance of business-critical software modules.
- Implementing and reviewing application functionality according to defined requirements.
- Working with SQL queries to support data-driven workflows.
- Collaborating with team members through structured development and delivery processes.
- Supporting code organization, traceability, and deployment preparation.
